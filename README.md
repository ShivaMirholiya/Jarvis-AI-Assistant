# 🤖 Jarvis AI Assistant

A Python-based AI assistant built while learning Python.

## ✨ Features

- 💬 AI Chat
- 🌦 Weather Information
- 📰 Latest News
- 🎤 Voice Interaction

## 🛠 Technologies Used

- Python
- OpenAI API
- NewsAPI
- python-dotenv

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/ShivaMirholiya/Jarvis-AI-Assistant.git
```

Go to the project folder:

```bash
cd Jarvis-AI-Assistant
```

Install the required packages:

```bash
python -m pip install -r requirements.txt
```

## 🔑 Environment Variables

Create a `.env` file and add:

```env
OPENAI_API_KEY=your_openai_api_key
NEWS_API_KEY=your_newsapi_key
```

## ▶️ Run the Project

```bash
python main.py
```
